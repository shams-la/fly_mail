# Fly Mail

Fly Mail a python GUI application with modern UI for sending emails

Download -> link will be added very soon!

What things you need to install


## Features
- Dark Mode
- Interactive Design
- Stores Last Changes
- Report Bug In New Window
- Processing Loader
- Full Error Handling
